🌐 科学上网完整指南

本文详细介绍了从桌面端到移动端的完整科学上网解决方案，包含主流代理工具的安装配置和使用方法。

💻 桌面端使用教程

代理工具下载及安装

以下是各内核的主流代理工具，可根据个人喜好选择：

工具名称 内核 下载地址
Clash Verge Clash Meta 下载链接
V2rayN Xray 下载链接
Hysteria sing-box 下载链接
ClashX.Meta Clash Meta (Mac) 下载链接

温馨提示：如果Github访问困难，可安装Github 增强 - 高速下载插件加速下载，或安装Watt Toolkit挂载代理访问Github

Clash Verge 详细使用指南

语言设置

Clash Verge 默认支持中文界面。如显示英文，请进入设置(Setting) → Language → 选择中文。

订阅配置流程

  1. 进入订阅页面

    • 点击左侧菜单栏的配置选项

  2. 导入订阅链接

    • 在顶部输入框粘贴你购买的机场订阅链接
    • 点击导入按钮

  3. 处理导入问题
    如遇下载失败，可能是因为需要代理才能访问订阅链接。解决方案请查看机场教程内的机场订阅Notion笔记

  4. 配置文件管理

    • 导入成功后若出现 Remote File 配置
    • 右键点击配置选择编辑信息，可修改名称和更新间隔

  5. 激活订阅

    • 点击导入的订阅文件，卡片左侧显示如图所示数显即为该订阅使用中

节点选择与代理设置

激活订阅后，进入代理页面查看所有可用节点：

节点分组说明：

  • 手动选择：包含所有可用节点，可手动测试选择
  • 自动选择：机场自动分配最优节点
  • 故障转移：主节点失效时自动切换备用节点

延迟测试与排序：

  • 点击延迟测试图标测量节点响应时间
  • 使用排序图标按延迟优化节点列表
    代理模式选择：
    可通过页面右上角选择代理模式
模式 说明 适用场景
规则模式 根据预设规则智能分流 日常使用
全局模式 所有流量都走代理 特殊需求
直连模式 完全不使用代理 国内网站

重要：完成配置后记得在设置中开启系统代理

📱 移动端使用教程

安全提醒

避免使用的VPN应用：

  • 老王VPN、极光VPN、蓝灯VPN
  • 小牛VPN、黑洞加速器、旋风加速器
  • 雷霆加速器、蜜蜂加速器等

这些应用多数为国内团队开发，存在安全风险，已有用户因使用此类应用受到处罚记录。

Android 端配置

应用下载：

  • 网盘下载(提取码:4wko)
  • Surfboard 官网
  • ClashMeta 官网
  • Hysteria 官网

版本选择指南：

  • foss：开源内核版
  • premium：闭源内核版（功能更强大，推荐）
  • 处理器架构选择：
    • Universal：通用版（兼容性好）
    • arm64-v8a：64位ARM处理器
    • armeabi-v7a：32位ARM处理器

Surfboard 使用教程

配置步骤：

  1. 导入订阅：配置页面 → URL导入或一键订阅
  2. 节点测试：代理页面测试延迟，选择最优节点
  3. 启动服务：主页启动代理，状态栏显示钥匙图标即连接成功

iOS 端配置

前提条件：需要美区Apple ID

获取方式：

  • 自行注册（注册教程）
  • 电商平台购买（搜索关键词：美区ID、ios美区账号）

推荐应用：

  • Shadowrocket(小火箭) - $2.99
  • Quantumult X - $7.99
  • Stash - $2.99
  • Loon - $4.99
