🔗 订阅链接使用方法

不同机场的操作流程基本相似，这里以ACYun机场为例演示完整流程。

购买订阅套餐

  1. 登录后点击左侧「购买订阅」

  2. 选择合适的套餐点击「立即订阅」

免费套餐说明：部分机场提供免费订阅，通常有两种形式：

  1. 注册后自动获得基础订阅

  2. 需要手动购买0元套餐

导入订阅链接

购买完成后，进入仪表盘查看订阅信息：

  • 一键订阅：支持Clash for Windows等软件快速导入

  • 手动导入：复制订阅链接到Clash Verge等需要手动配置的软件

如遇任何操作问题，欢迎在评论区留言，我会及时回复解答

💰 费用与计费机制

流量倍率详解

倍率定义：节点流量的计量系数，计算公式为：==系统计费流量 = 实际使用流量 × 节点倍率==

实际示例：

  • 下载100MB文件，使用不同倍率节点的计费对比：
节点倍率 计费流量 实际消耗
×1 100MB 100MB
×3 300MB 100MB
×5 500MB 100MB

倍率选择策略

机场设置倍率的官方解释：

  • 🚀 成本分摊：中转节点、BGP、IPLC专线涉及多台服务器，流量成本翻倍

  • 💎 质量保障：优质服务器和线路投入更高成本

选择建议：

  • 日常使用：选择低倍率节点节省流量

  • 特殊需求：高倍率节点适合对速度稳定性要求高的场景

🌐 节点类型完全解析

不同节点类型的连接路径对比：

图例说明：

  • 🔴 红色线路：直连节点

  • 🔵 蓝色线路：中转节点

  • 🟢 绿色线路：IPLC国际专线

直连节点

路径：用户 → 运营商宽带 → GFW审查 → QoS限速 → 海外服务器

特点：

优点 缺点
💰 价格便宜 🐢 速度较慢
设置简单 🔒 易被封锁
倍率较低 稳定性一般

中转节点

路径：用户 → 国内中转服务器 → 海外服务器

优势对比：

  • ⚡ 速度更快：优化路由路径

  • 📉 延迟更低：减少网络跳数

  • 🔐 更安全：多层加密保护

  • 🛡️ 更稳定：不易被GFW检测

中转线路类型详解

线路类型 全称 特点 适用场景
BGP专线 Border Gateway Protocol 智能路由选择，隧道加密 日常上网，平衡性价比
IPLC专线 International Private Leased Circuit 绕过GFW，极低延迟 视频会议，在线游戏
IEPL专线 International Ethernet Private Line 企业级品质，高可靠性 企业用户，专业需求
CN2线路 Chinatelecom Next Carrier Network 电信精品网络，优质QoS 对稳定性要求高的用户
CN2 GIA Global Internet Access CN2顶级产品，全程高速节点 高端用户，极致体验

技术说明：

  • GFW：Great Firewall 防火长城

  • QoS限速：服务质量机制，民用线路优先级较低

  • CN2 GIA：全程使用59.43高速节点，提供最优性能
